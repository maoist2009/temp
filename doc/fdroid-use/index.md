## fdroid软件源使用说明                 
- 有海外代理/海外vpn的情况下，最好不使用国内镜像源直接开启代理/vpn后使用，最大程度保证隐私                   
- 没有海外代理/vpn的情况下，可以使用下面的方法：   
- 打开下面网站：      
https://help.mirrors.cernet.edu.cn/fdroid/      
如果无法加载成功可以打开下面网站：      
https://codeberg.org/Purify/F-Droid_Mirror      
- 打开上面的网站，选择想要的国内站点镜像源，然后复制链接。      
比如南京大学源    
https://mirror.nju.edu.cn/fdroid/repo/?fingerprint=43238D512C1E5EB2D6569F4A3AFBF5523418B82E0A3ED1552770ABB9A9C9CCAB      

- 打开fdoid软件      
-fdroid软件设置-储存库-右下角“+”添加-粘贴镜像源-勾选确认 -回到储存库-点击F-droid图标-点击官方镜像关闭所有官方镜像-保持用户镜像打开-回到软件首页-点击右下角最新-下滑软件主界面刷新软件列表并等待更新完成-给予软件通知权限后，通知栏可以看到下载和更新进度（如果进度卡住退出后台后重启软件然后重新刷新，如果无速度可以换一个源）  
- 最后下载想要的软件。

#### 二.github免代理下载软件教程       
1. 下载火狐浏览器国际版或者其分支     
- [火狐浏览器安卓版](https://ftp.mozilla.org/pub/fenix/releases/)-选择版本号最大的，下载arm64 v8a版   
- [火狐浏览器电脑版](https://www.mozilla.org/en-US/firefox/all/desktop-release/)          

2. 安装浏览器[油猴子插件](https://addons.mozilla.org/zh-CN/firefox/addon/violentmonkey/?utm_source=addons.mozilla.org&utm_medium=referral&utm_content=search)   ,使用火狐浏览器访问并安装插件，同时安装[翻译插件twp](https://addons.mozilla.org/zh-CN/firefox/addon/traduzir-paginas-web/?utm_source=addons.mozilla.org&utm_medium=referral&utm_content=search)  -默认翻译引擎为谷歌，需要手动切换为bing  

3. 安装加速脚本
用火狐浏览器访问此[脚本1-github加速下载](https://greasyfork.org/zh-CN/scripts/398278-github-%E9%95%9C%E5%83%8F%E8%AE%BF%E9%97%AE-%E5%8A%A0%E9%80%9F%E4%B8%8B%E8%BD%BD)            
- 如果github网页无法正常访问，可以使用[脚本2-github加速访问](https://greasyfork.org/zh-CN/scripts/462685-github%E5%8A%A0%E9%80%9F%E8%AE%BF%E9%97%AE%E6%8F%92%E4%BB%B6)  

#### 另外一种方式
找网页代理加速，把github网址复制到加速站解析为加速网页后访问，缺点是每一次点击跳转，都要把新网页地址复制到加速站再次解析为新加速地址，这种非智能 
加速站实例：
https://xuc.xi-xu.me/    
https://ghproxy.link/         

3. 如何检索github    
- 使用搜索引擎检索，比如[微软bing](https://cn.bing.com/),或者[开源搜索引擎pub](https://search.pub.solar/),检索  xxxxx github  （xxx为项目或软件名）   
- [github网站内搜索](https://github.com/)        

4. github的打包好的软件下载，一般是在页面内找release释放
