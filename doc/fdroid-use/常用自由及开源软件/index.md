- [shizuku]-免root获取安卓高权限，github或者官网下载，教程也在官网 ，[图文教程可参考这里](/doc/fdroid-use/%E5%B8%B8%E7%94%A8%E8%87%AA%E7%94%B1%E5%8F%8A%E5%BC%80%E6%BA%90%E8%BD%AF%E4%BB%B6/shiziku/)

- [app manager](https://f-droid.org/en/packages/io.github.muntashirakon.AppManager/)- 可以在fdroid上下载 ，功能比较齐全，ops权限管理（汉化差） ，屏蔽软件内的跟踪  ，应用安装器可以安装谷歌应用商店的软件包格式比如apkx和apks  等等 ，需要root激活

- app ops -权限管理，在其官网下载，汉化程度最高界面简洁但是非开源，因此只适合放在私有软件所在用户空间内管理私有软件使用  ，如果要在自由开源类原生安卓系统使用请关闭其联网权限 ，需要shizuku激活           
- magisk /狐狸面具/ksu内核root  root工具（github下载）

https://github.com/tiann/KernelSU     
4. Apatch内核root工具（支持机型较kernelsu多，功能相似，root隐藏较好，防止部分国产私有软件检测到root不让使用，带root密码）

### 十. 加密/伪装设备信息/隐藏 /工作空间/沙盒 /隐藏伪装地理    
-  *EDS Lite/ SSE/Openkeychain 文件加密  
-  *Extirpater 剩余空间擦除防止被恢复     
-  Duress 错误密码登陆重置手机 （程序自毁）
- [Locker](https://f-droid.org/zh_Hans/packages/net.zygotelabs.locker/)  指定允许的数量
尝试解锁您的手机。超过设置次数就重置手机，但是代码较旧已经停止维护，不确定在高版本安卓的兼容性如何   
- sentry 错误密码n次重置手机       
-  shelter-工作空间，独立用户空间隔离主用户文件和软件   
-  [炼妖壶](https://github.com/oasisfeng/island/)  --github上下载，作用同shelter 
-  [meta 元萝卜沙盒国际版 ](https://github.com/Katana-Official/Meta) ---github上下载，，沙盒化隔离安装应用，支持启动加密和错误密码自毁，无需root         
使用方法推荐：可以在手机里先启用多用户/隐私空间/工作空间，然后在里面安装元萝卜沙盒，然后元萝卜里面安装xmpp，电报等政治风险软件     
元萝卜沙盒内和环境和真机独立，内部软件包名和安装不受系统管家/权限控制和检测，除非是屏幕识别软件服务，可以设置设备伪装，设置启动密码和自毁密码，如果被逼问密码告诉自毁密码自毁沙盒内所有程序和历史。    
该软件有两种模式，第一种是直接从本地安装软件到沙盒，第二种兼容性更高，使用的是应用分身的方式，如果第一种模式无法运行就使用第二种。  
目前这种使用方法是学习成本和使用难度最低的，在任意安卓手机上都可以使用，无需root和解bl锁，也没有安卓虚拟机那样需要wifi调试解锁后台限制。   
不过需注意此软件可以免费自由使用，所用sdk不开源，无需身份绑定，不再国内应用市场发布。
