### Linux常用开源软件推荐-侧重于带图形化软件                      
- 代理/vpn软件- [v2rayn](https://github.com/2dust/v2rayN),[v2raya](https://v2raya.org/),[hiddify](https://github.com/hiddify/hiddify-app),[clash-verage-rev](https://github.com/clash-verge-rev/clash-verge-rev) ，[flclash](https://github.com/chen08209/FlClash),[tor-browser](https://www.torproject.org/download/)，迷雾通（需要先安装flatpak包管理器），protonvpn....              
- 中文输入法-fcitx5 或 ibus（都需要安装软件本体后再安装中文输入插件）             
- 软件应用隐藏工具：  xfce4-appfinder（点击自己的应用，鼠标右键有隐藏功能，记住软件英文名后选择隐藏，然后在终端里输入英文名启动软件，隐藏后的软件不会出现在开开始菜单栏中）       
- 应用商店  pamac-manager（arch图像化软件管理工具，支持官方仓库和自定义仓库的软件安装的卸载，软件设置内可以启用aur用户仓库，刷新后可使用），gnome-soft（gnome默认应用商店，安装插件后可以支持flatpak仓库），dicover（kde桌面的默认应用商店）     

- 网络浏览器 firefox，firefox-nightly（火狐浏览器每夜版，支持离线中文翻译），GNU/Icecat(火狐浏览器自由版) ，tor-broswer，chromium，brave，       
- 下载器 
[ ab downloader](https://github.com/amir1376/ab-download-manager)
,[gopeed](https://www.gopeed.com/zh-CN)（支持磁力链接），[motrix](https://motrix.app/)（支持磁力）      
- 文件解压缩 -peazip，file-roller       
- 办公软件套件-libreoffice（包含pdf处理软件），onlyoffice         
- 阅读器-   koodoreader，calibre（有图书翻译功能插件）
- 文档扫描 ：simple-scan   ，
- 文件加密-veracrypt,kgpg    
- 密码管理  -在线密码管理-Bitwarden，离线本地密码管理-kwalletmanager（kde桌面自带）         
- 文件查看 nautilus（gnome桌面自带，可独立安装）
- 磁盘管理/磁盘分区加密：gnome-disks
- 垃圾清理/进程管理/启动项管理/文件擦除和粉碎-bleachbit，stacer         
- 元数据清理（清除图片和其他文件中的用户跟踪数据）： metadata-cleaner       
- 启动盘多系统引导制作工具-ventoy      
- 图片视频格式转换：Open-Format-Conversion，Open格式转换 是一款专为 Linux 系统开发 的 图形化格式转换工具，旨在为用户提供轻量、高效格式转换解决方案。项目灵感源自 Deepin 论坛网友需求——打造一个 Linux 原生 GUI 交互 的开源程序。  地址   https://github.com/mrarub/Open-Format-Conversion/      
- 图片处理-gimp   
- 视频处理/制作-Kdenlive       
- 音频处理/制作    -   Audacity     
- 图片查看-eog     
- ocr图片处理软件-gImageReader  （arch官方软件仓库有）  ，用这个软件进行图片哦承认。识别需要安装orc开源引擎，参考[图片识别中文ocr引擎](https://github.com/gumblex/tessdata_chi)           
    
- 离线视频播放器-vlc,mpv      
- 免费在线视频播放器    ZY-Player （ 自定义播放源，需要在github或者搜索引擎找播放源  ）    ,freetube（youtube视频第三方开源软件，支持免登陆观看下载和免登录关注订阅以及第三方开源youtube源）         
- 离线音乐播放器-Rhythmbox        
- 免费在线音乐播放器-musicfree（可自定义音乐源-播放源添加参考-https://github.com/Huibq/keep-alive             ）,      algermusic（网易云第三方），       
- 代码书写/md/网页制作-[vscodium](https://vscodium.com/),(微软vscode的第三方开源版本-不包含跟踪)     
- 杀毒软件-clamtk      
- 防火墙-firewalld    
- 虚拟机-virtualbox,kvm,gnome boxes      ，docker      
-  安卓软件开发 android-studio       
- 安卓连接电脑的adb工具-：android-tool          
- 沙盒-firejail          
- flatpak的权限管理软件（flatpak是自带沙盒功能的跨linux发行版的通用软件包格式）-flatseal                          
- 开源ai本地部署- [ollama](https://ollama.com/),支持谷歌，deepsink等多种ai引擎                
- 端对端加密聊天软件   xmpp客户端有-   gajim,dino     。      matrix客户端有-element      
- 视频会议                
-  邮箱- thunderbird      
- 远程桌面 - rustdesk                 
- [gdm-setting](https://github.com/gdm-settings/gdm-settings) (gnome桌面环境的gdm设置管理软件，可以设置登陆时隐藏默认用户名和默认登陆用户，kde桌面通过桌面登陆设置更换主题也可以办到)
